So, this is my little vim handbook, entirely written in vim.
	1. Movement in vim:
	What which key does:
	  1. "Esc" - Entering command mode. Self explanatory (I hope so).
	  2. "i" - Entering insert mode. In this mode you are able to write in file.
	  3. "v" - Entering visual mode. In this mode you can select text (for copying, cutting, etc.).
	1. Basic operations on file:
	All you need to acually do something in vim:
	  1. :w - Writes file.
	  2. :q - Quits file.
	  3. :q! (shift + ZQ) - Force quits file (quits without saving).
	  4. :wq (:x, shift + ZZ) - Writes and quits file.
	  5. :w <file name> - Writes to a new file with inputed name.
	  6. y - Copy.
	  7. p - Paste.
	  8. d - Cut.
